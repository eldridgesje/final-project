Analysis of Big 5 Personality Test

Typically, unsupervised algorithms, like K-means Clustering, make inferences from datasets using only input vectors without referring to known or labelled, outcomes. But what if one tested a well known and respected set of data using the k-means. Would the clustered outcomes show predictable correlations among the features tested? Might variations in some correlations indicate that one could improve the known model by emphasizing one data type over another? We located a dataset based on The Big Five personality model and determined to explore these possibilities using K-means testing.
