Analysis of Big 5 Personality Test






Feature Varience Over The Clusters

Several demographic features were considered with respect to how their data might have varied over the five data clusters.   The demographic features tested included gender, age, race, and left/right handedness.   For each of these featuresâ€™ hypothesis testing was completed.

While the specific ANOVA numbers varied, the overall results on all demographic features were like those demonstrated by the "Age" boxplot to the left.  Means and data dispersions were very similar across the clusters.   ANOVA tests all revealed high pvalue scores.   So, in all cases of demographic data the Null Hypothesis failed because all pvalues where significantly higher than .05.

Hypothesis Testing

  • Gender:  pvalue  =  0.6304
  • Age:   pvalue  =  0.5941
  • Race:   pvalue = 0.4508
  • Handedness:   pvalue = 0.8865

Null Hypothesis (H0):  Clusters are NOT similar across features such as gender, age, handedness, or native english speaking.

Conclusion: We fail to reject H0 because p values are all >>.05.   Features ARE similar across all clusters.
